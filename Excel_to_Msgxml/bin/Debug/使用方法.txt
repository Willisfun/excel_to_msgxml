==========================
Excel 轉 MSGXML 使用說明
==========================

1. 開啟程式，選擇要轉換的 Excel 檔案。

2. 輸入資料開始位置：
   - Row：資料開始的列號。
   - Column：資料所在的欄號。

   例如資料位於 B4：
       Row = 4
       Column = 2

   欄位對照：
       A = 1
       B = 2
       C = 3
       D = 4
       ...

3. 工作表規則：

   (1) 繁體中文
       使用 Excel 第一個工作表。

   (2) 簡體中文
       工作表名稱必須包含「簡」。

   (3) 英文
       工作表名稱必須包含「EN」。

   範例：

       M Code
       M Code 簡
       M Code EN

4. 若工作表名稱不符合以上規則，
   請先修改 Excel 的工作表名稱後再執行程式。

5. 程式會從指定的 Row、Column 開始讀取資料。

6. Excel 空白儲存格會自動跳過，
   MSGXML 的 Row 會依序連續寫入，不會因空白而保留空的 Row。

7. 轉換完成後，MSGXML 會輸出到程式所在資料夾。
